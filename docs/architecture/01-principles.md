# Architecture Principles

## 1. Cognition-First Behavior
- Prioritize cognitive processing and understanding in all engineering decisions.
- Ensure that interactions and user experiences reflect a deep understanding of cognitive processes.

## 2. Curated Memory
- Implement memory systems that allow for effective information retrieval and management.
- Focus on organizing and structuring information to enhance usability and performance.

## 3. Implementation Constraints
- Identify and document all technical constraints facing the implementation of systems.
- Consider factors such as resource limitations, platform capabilities, and user accessibility in all design decisions.

## Conclusion
These principles guide the development and evolution of architecture in cognitive AI systems, ensuring alignment with cognitive behaviors and effective resource management.
